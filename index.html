Casos de uso

Para depurar dados locais sem abrir mão de contexto.

O DataSidekick ajuda quando o estado da aplicação está no navegador e você precisa entender, ajustar ou compartilhar esse estado com rapidez.

Editar JSON no LocalStorage

Abra objetos aninhados em uma árvore visual, altere campos específicos e salve sem reescrever strings compactadas manualmente.

Inspecionar SessionStorage

Veja rascunhos, fluxos temporários, filtros e estados de sessão da aba atual sem misturar dados de outras origens.

Exportar dados locais em JSON

Gere backups, compartilhe cenários de teste e restaure estados conhecidos durante desenvolvimento, QA ou suporte.

Testar feature flags e preferências

Ajuste flags, temas, preferências e caches locais para validar comportamentos sem limpar todo o navegador.

Privacidade

Seus dados ficam no seu navegador.

Sem coleta de dados

A extensão não vende, transmite ou compartilha dados do usuário. O acesso ao storage acontece localmente para exibição e edição solicitadas pelo usuário.

Sem código remoto

O HTML, CSS e JavaScript são empacotados dentro da extensão. O DataSidekick não carrega scripts hospedados remotamente.

Permissões justificadas

As permissões são usadas para abrir o painel lateral, detectar a aba ativa e executar código local no contexto da página atual.

Como o DataSidekick acessa seus dados

O DataSidekick não solicita acesso permanente a todos os sites durante a instalação.

Quando você abre o painel em uma página, a extensão identifica a origem atual, por exemplo localhost:3000 ou app.example.com. Para ler e editar localStorage e sessionStorage, o DataSidekick solicita permissão apenas para aquela origem.

Esse acesso é usado somente para:

  • listar chaves do LocalStorage e SessionStorage;
  • exibir valores no painel lateral;
  • editar valores quando você salva alterações;
  • importar ou exportar dados em JSON;
  • excluir chaves quando você confirma a ação.

O DataSidekick não envia seus dados para servidores externos, não usa código remoto e não compartilha dados com terceiros. Tudo acontece localmente no navegador.

Como funciona

Do storage ao JSON editável em segundos.

  1. Abra o painel lateral O DataSidekick aparece ao lado da página atual.
  2. Escolha LocalStorage ou SessionStorage A extensão detecta a origem da aba ativa.
  3. Busque e edite Edite valores simples inline ou JSON em árvore visual.
  4. Salve, importe ou exporte Faça backup ou restaure dados em JSON localmente.
Perguntas frequentes

Dúvidas comuns antes de instalar.

O DataSidekick envia meus dados para servidores?

Não. Os dados de LocalStorage e SessionStorage são lidos e editados localmente no navegador. A extensão não envia esses dados para servidores externos.

A extensão acessa todos os sites?

Não durante a instalação. Quando você abre o painel em uma página, o DataSidekick solicita permissão apenas para a origem atual.

Posso editar objetos JSON aninhados?

Sim. Valores JSON válidos são exibidos como uma árvore visual editável para facilitar alterações pontuais.

Para quem o DataSidekick foi criado?

Para desenvolvedores, QAs e pessoas técnicas que precisam depurar, editar, importar ou exportar dados locais de aplicações web.
