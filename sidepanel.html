Ajuda

Como começar

  1. Abra uma página ou aplicação local em uma aba do Chrome.
  2. Clique no DataSidekick para abrir o painel lateral.
  3. Permita o acesso à origem atual quando o painel solicitar.
  4. Escolha entre LocalStorage e SessionStorage no topo.

Editar dados

Selecione uma key na lista para abrir o editor. Valores JSON válidos aparecem como árvore editável; valores simples aparecem como texto.

Depois de alterar um valor, clique em Salvar ou use Ctrl+S / Cmd+S.

Organizar a lista

  • Use a busca para localizar keys ou conteúdos.
  • Favorite keys importantes com o ícone de estrela.
  • Oculte keys ruidosas ou específicas pelas ações da lista.
  • Reexiba keys ocultas em Configurações.

Importar e exportar

Exportar gera um arquivo JSON com metadados da origem, tipo de storage e itens atuais. Importar aceita esse formato ou um objeto JSON simples de chave e valor.

Privacidade

O DataSidekick roda localmente no navegador. Ele lê e altera apenas o storage da aba ativa, depois que você concede permissão para aquela origem.
